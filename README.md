# python-sandbox

## Вимоги

- Python 3 встановлено у системі
- Для скрипта `hello_world_turtle.py` потрібне графічне середовище, а також встановлений модуль Tkinter (`tk`), оскільки бібліотека `turtle` використовує його для графіки (у більшості систем входить до складу стандартної бібліотеки, але іноді потребує окремої установки).
    - Перевірити наявність можна командою: `python3 -m tkinter`
    - Для Linux/Ubuntu, якщо потрібен модуль: `sudo apt install python3-tk`

## Як запустити

1. Відкрий термінал у папці з файлами.
2. Запусти команду:

```bash
python3 main.py
```

## Опис скриптів

- **main.py**  
  Виводить список усіх `.py`-файлів у цій папці (крім себе), дозволяє вибрати номер і запускає вибраний скрипт.

- **hello_world_console.py**  
  Виводить у консоль повідомлення "Hello World!".

- **hello_world_turtle.py**  
  Відкриває графічне вікно й малює лінію завдовжки 100 пікселів із допомогою модуля `turtle`.
